jQuery dimensions


With jQuery it is easy to work with dimensions of elements and browser window


jQuery dimensions methods


jQuery has several important methods for working with dimensions:

  • width()
  • height()
  • innerWidth()
  • innerHeight()
  • outerWidth()
  • outerHeight()

jQuery innerWidth() and innerHeight() methods


The innerWidth() method returns the width of an element (includes padding) The innerHeight() method returns the height of an element (includes padding) The following example returns the inner width/height of a specified <div> element:


Example


            $('button').click(function() {
                var txt = "";
                txt += "Inner width is: " + $('#div1').innerWidth() + "<br>";
                txt += "Inner height is: " + $('#div1').innerHeight() + "<br>";
                $("#div1").html(txt);
            })

jQuery outerWidth() and outerHeight() methods


The outerWidth() method returns the width of an element (including padding and border). The outerHeight() method returns the height of an element (includin padding and border). The following example returns the width/height of a spcified <div> element


Example


            $('#btn1').click(function() {
                var txt = "";
                txt += "Outer width is: " + $("#div1").outerWidth() + "<br>";
                txt += "Outer height is: " + $("#div1").outerHeight();
                $("#div1").html(txt);
            })

The outerWidth(true) method returns the width of an element (includs padding, border and margin). The outerHeight(true) method returns the width of an element (includs padding, border and margin).


jQuery more width and height


The following example returns the width and height of the document (the HTML document) and window (the browser viewport):


Example


            $("button").click(function() {
                var txt = "";
                txt += "dimensions of the document are: " + $(document).width() + "x" +
                        $(document).height() + "\n";
                txt += "dimensions of the window are: " +$(window).width() + "x" +
                        $(window).height() + '.';
                $("#div1").html(txt);
            });

The following example sets the width and height of a specified HTML element:


Example


            $("button").click(function() {
                $("#div1").width(500).height(500);
            });
