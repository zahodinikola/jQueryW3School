jQuery - remove elements


With jQuery it is easy to remove existing HTML elements


Remove elements/content


To remove elements and content there are mainly two jQuery methods:

  • remove() - removes the selected HTML elements (and its child elements);
  • empty() - removes the child elements from the selected element

jQuery remove() method


The jQuery remove() method removes selected elements and all its child elements


Example


			$("#div1").remove();

jQuery empty method


The jQuery empty() method removes the child elements from the selected elements


Example


			$("div").empty();


Filter the elements to be removed


The jQuery remove() method also accepts one parameter, wich allows you filter the elements to be removed. The parameter can be any of the jQuery syntaxes. The following example removes all >p< elements with class="test".


Example


			$("p").remove('.test');

Next example removes all >p< elements with classes "test" and "demo". 

			$("p").remove(".test, .demo");
