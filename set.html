jQuery - set content and attributes


Ser content - text(), html() and val()


We will use the same three methods from the previous page to set content


  • text() - sets or returns the text content of selected element
  • html() - sets or returns content of selected element (including html)
  • val() - sets or returns the value of form fields

The following example demonstrates how to set content with text(), html() or val() methods.


Example



						$(document).ready(function() {
							$("#btn1").click(function() {
								$("#test1").text("hello, buddy!");
							});
							$("#btn2").click(function() {
								$("#test2").html("'hello, buddy!'");
							});
							$("#btn3").click(function() {
								$("#test3").val('hi, there');
							});
						});

A callback function for text(), html() and val() methods


All of the three jQuery methods above: text(), html() and val() also come with a callback function. The callback function has two parameters: the index of the current element in the list of elements selected and the original (old) value. You then return the string you wish to use as the new value from the function. The following example demonstrates text() and html() with a callback function.


Example


			$("#btn1").click(function() {
				$("#test1").text(function(i, oldValue) {
					return "old value: " + oldValue + " new text: index " + i + ".";
				});
			});
			$("#btn2").click(function() {
				$("#test2").html(function(i, oldValue) {
					return "Old value: " + oldValue + "new text: index" + i + ".";
				});
			});

Set attributes - attr()


The jQuery attr() method is used to set/change attribute values


The following example demonstrates how to set/change the value of href attribute in link


Example


			$('button').click(function() {
				$('#w3s').attr("href", "http://w3school.com");
			});

The attr() method also allows you to set multiple attributes at the same time

The following example demonstrates how to set both the href and title attributes at the same time.


Example


			$("button").click(function() {
				$("#w3s").attr({
					"href": "https://www.w3schools.com/jquery",
					"title": "w3school jQuery tutorial"
				});
			});


A callback function for attr()


The jQuery mathod attr() also come with a callback function. The callback function has two parameters: the index of the curent element in the list of elements selected and the original (old) attribute value. You then return the string you wish to use as the new attribute value from the function.

The following example demonstrates attr() with a callback function 


			$("button").click(function() {
				$("#w3s").attr("href", function(i, oldValue) {
					return oldValue + "/jquery";
				});
			});
