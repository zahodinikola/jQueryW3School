jQuery hide() and show()

With jQuery you can hide and show elements with the hide() and show() methods

Example

			$("#hide").click(function(){
				$("p").hide();
			});
			$("#show").click(function(){
				$("p").show();
			});

Syntax

			$(selector).hide(speed, callback);
			$(selector).show(speed, callback);

The optional parameter speed specifies the speed of hide/show and can take values "slow", "fast" and milliseconds. The optional callback parameter is a function to be executed after the hide() or show method completes (more about callback functions later).

Example

			$("button").click(function() {
				$("p").hide(1000);
			});

jQuery toggle()


With jQuery you can toggle between hide() and show() methods with the toggle() method. Shown elements are hidden and hidden are shown: 


			$(document).ready(function() {
				$("pre").click(function() {
					$("p").hide(1000);
				});
			});

jQuery effects - fading


With jQuery you can fade elements in and out of visibility

jQuery fading methods:

			jQuery has the following methods:
				fadeIn();
				fadeOut();
				fadeToggle();
				fadeTo();

jQuery fadeIn() Method

The jQuery fadeIn() method is used to fade in a hidden element

Syntax

			$(selector).fadeIn(speed, callback);

The optional speed parameter specifies the duration of the effect

It can take the fallowing values: "slow", "fast" or milliseconds.

The optional callback parameter is a function to be executed after the fading completes

The following example demonstrates the fadeIn() method with different parameters

Example

			$("button").click(function() {
				$("#div1").fadeIn();
				$("#div2").fadeIn(1000);
				$("#div3").fadeIn('fast');
				$("#div4").fadeIn('slow');
				$("#div5").fadeIn('2000', alert('123')); pay attention to calling alert - it's breaking fading
			});

jQuery fadeOut() method

The jQuery fadeOut() method is used to fade out a visible element

Syntax

			$(selector).fadeOut(speed, callback);

The optional speed parameter specifies the duration of the effect. It can take the fallowing values: 'fast', 'slow' or milliseconds. The optional parameter callback specifies the function to be executed after the fading completes. The fallowing example demonstrates the fadeOut() method with different parameters.

Example

			$(document).ready(function() {
				$("button").click(function() {
					$("#div1").fadeOut();
					$("#div2").fadeOut(1000);
					$("#div3").fadeOut('fast');
					$("#div4").fadeOut('slow');
					$("#div5").fadeOut(3000, func(params));
				});
			});

jQuery fadeToggle() method

The jQuery fadeToglle() method toggles between the fadeIn() and fadeOut() methods. If the elements are faded out, fadeToggle() will fade them in. If the elements are faded in, fadeToggle() will fade them out.

Syntax

            $(document).ready(function(){
                $(selector).event(function(){
                    $(selector).fadeToggle(speed, callback);
                });
            });

The optional speed parameter specifies the duration of the effect. It can take the fallowing values: 'fast', 'slow' or milliseconds. The optional callback parameter is a function to be executed after the fading completes. The following example demonstrates the fadeToggle() method with different parameters:

Example

            $(document).ready(function() {
                $("button").click(function() {
                    $("#div1").fadeToggle();
                    $("#div2").fadeToggle(3000);
                    $("#div3").fadeToggle('slow');
                });
            });

jQuery fadeTo() method

The jQuery fadeTo() method allows fading to a given opacity (value between 0 and 1).

Syntax

            $(selector).fadeTo(speed, opacity, callback);

The required speed parameter specifies the duration of the effect. It can take the following values: 'fast', 'slow' and milliseconds. The required opacity parameter in the fadeTo() method specifies fading to a given opacity (value between 0 and 1). The optional callback parameter is a function to be executed after the function completes.

Example

			$(document).ready(function() {
				$("button").click(function() {
					$("#div").fadeTo();
				});
			});
